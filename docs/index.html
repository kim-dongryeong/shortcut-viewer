What is Shortcut Viewer?

The pages above are static cheat sheets for each app's default shortcuts. The real tool is an interactive macOS & Windows app that merges shortcuts from every source on your machine — the system, running app menus, VS Code & Adobe keymaps, Karabiner/PowerToys, BetterTouchTool, Raycast and your own globals — onto one keyboard grid, then flags conflicts and finds free key combos. Open source, GPL-3.0.

Shortcut Viewer — Every Mac and Windows keyboard shortcut, on one keyboard grid.
